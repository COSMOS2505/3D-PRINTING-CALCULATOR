# S.Y.S.T.E.M. - 3D Printing Cost Dashboard 🚀

**S.Y.S.T.E.M.** (Smart Yield & Sustainable Technical Estimation Manager) is a professional desktop application designed for 3D printing enthusiasts and entrepreneurs to accurately calculate production costs and manage profit margins.

## 🧐 Why use S.Y.S.T.E.M.?

Pricing a 3D printed part is more than just material weight. This tool helps you account for:
* **Energy Consumption:** Based on your printer's wattage and local electricity rates.
* **Maintenance & Wear:** Automatic inclusion of machine wear-and-tear costs.
* **Labor & Extras:** Add service fees and packaging costs easily.
* **Real Profitability:** Instantly see your absolute profit and percentage margin.


---

## ✨ Features

* **Modern UI:** A sleek, dark-themed dashboard built with `CustomTkinter`.
* **Dynamic Calculations:** Real-time updates as you type.
* **Business Insights:** Color-coded profit indicators (Red for loss, Green for profit).
* **Standalone Executable:** No need to install Python to run the app.
